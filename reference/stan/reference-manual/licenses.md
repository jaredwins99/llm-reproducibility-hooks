# Licenses and Dependencies

Stan and its dependency libraries are distributed under open-source licenses that pose no restrictions on redistribution or application use.

## License Summary

**Stan** uses the BSD 3-clause license, meaning "developers or assignees hold copyright for each contribution."

**Boost** provides template metaprograms and numerical libraries under the Boost Software License 1.0, with copyright held by respective package developers.

**Eigen** handles matrix operations via the Mozilla Public License 2.0, with joint developer copyright ownership.

**SUNDIALS** enables differential equation solving through the BSD 3-clause license, with copyright held by Lawrence Livermore National Security Lab.

**Threaded Building Blocks (TBB)** supports parallel computation via the Apache License 2.0, with copyright owned by Intel Corporation.

**Google Test** provides unit testing functionality through the BSD 3-clause license, with copyright held by Google, Inc. (not required for model execution).

The document emphasizes that these licenses contain "no copyleft provisions requiring applications of Stan to be open source if they are redistributed," offering flexibility for developers integrating Stan into proprietary or closed-source systems.
