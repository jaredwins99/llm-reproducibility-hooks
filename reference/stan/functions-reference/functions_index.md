# Stan Functions Index

## Overview

This is the alphabetical index of all Stan functions documented in the Functions Reference (Version 2.38). Each function links to its detailed documentation in the relevant chapter.

The index covers the complete set of Stan built-in functions organized alphabetically, including:

- Basic mathematical functions (abs, acos, asin, atan, etc.)
- Probability distributions (bernoulli, beta, binomial, cauchy, normal, etc.)
- Matrix/vector operations (append_col, append_row, block, cholesky_decompose, etc.)
- Statistical functions (mean, variance, sd, quantile, etc.)
- Linear algebra (eigenvalues_sym, eigenvectors_sym, singular_values, svd, etc.)
- Special functions (bessel, gamma, digamma, trigamma, etc.)
- Higher-order functions (ode_rk45, ode_bdf, integrate_1d, reduce_sum, etc.)
- Transform functions (simplex_constrain, ordered_constrain, etc.)
- Array operations (sort_asc, sort_desc, reverse, append_array, etc.)
- Complex number functions (to_complex, get_real, get_imag, conj, fft, etc.)
- Sparse matrix operations (csr_matrix_times_vector, csr_extract_w, etc.)
- Hidden Markov model functions (hmm_marginal, hmm_latent_rng, etc.)
- GP covariance functions (gp_exp_quad_cov, gp_matern32_cov, etc.)

For complete function signatures and descriptions, see the individual chapter files in this directory.
