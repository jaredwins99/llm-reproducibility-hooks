## Main Content

### Document Purpose
This guide serves as the user manual for CmdStan, described as "the command-line interface to the Stan statistical modeling language."

### Core Functionality
The document states CmdStan enables users to:

- "do inference on data, producing an exact or approximate estimate of the posterior"
- "generate new quantities of interest from an existing estimate"
- "generate data from the model according to a given set of parameters"

### Benefits Highlighted

**Latest Stan Access**: "With every new Stan release, there is a corresponding CmdStan release, therefore CmdStan provides access to the latest version of Stan"

**Efficiency**: CmdStan offers "the lightest memory footprint" among Stan interfaces and "the fewest dependencies"

**Compatibility**: Output generates in CSV format for post-processing flexibility
